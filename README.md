# Razorpay Clone 

Created this clone website of **Razorpay** using Tailwindcss


_[Click Here to Visit the Main Razorpay Website](https://razorpay.com/)_

## Installation
Install my project with npm

```
npm i
npm start
```

## Tech Stack

Tech stack used here is **`HTML`** and **`Tailwindcss`**
